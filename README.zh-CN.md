CyberCat 是一款 Windows 桌面应用，把 AI 聊天、语音录制和语音播报整合到一起。

## 安装

1. 打开最新的 GitHub Release 页面。
2. 下载 [CyberCat-setup-0.1.0.exe](https://github.com/zixiCat/CyberCat/releases/download/v0.1.0/CyberCat-setup-0.1.0.exe)。
3. 双击安装包，并按提示完成安装。
4. 从桌面快捷方式或开始菜单启动 CyberCat。

就这些，不需要你再手动安装别的东西。

## 你可以做什么

- 在桌面窗口里和 AI 助手聊天
- 录制语音并直接发送
- 收听应用播报的回复内容
- 使用快捷键更快完成操作

## 常用快捷键

- 按住 `z+x` 录音，确认后再发送
- 按住 `x+c` 录音并立即发送
- 按 `Ctrl+Shift+0` 朗读当前选中的文字

## 常见问题

- 如果 Windows 弹出安全提示，在确认你信任该版本后继续安装
- 如果安装后应用没有打开，先重启 Windows 再试一次
- 如果快捷键没有反应，检查是否有其他应用占用了相同按键
