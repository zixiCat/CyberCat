CyberCat 是一款 Windows 桌面应用，把 AI 聊天、语音录制和语音播报整合到一起。

## 安装

1. 打开最新的 [GitHub 发布页面](https://github.com/zixiCat/CyberCat/releases)。
2. 下载最新的 CyberCat-setup 安装包。
3. 双击安装包，并按提示完成安装。
4. 从桌面快捷方式或开始菜单启动 CyberCat。

就这些，不需要你再手动安装别的东西。

## 你可以做什么

- 在桌面窗口里和 AI 助手聊天
- 录制语音并直接发送
- 收听应用播报的回复内容
- 使用快捷键更快完成操作

## Bilibili / BBDown

CyberCat 提供用于 BBDown 工作流的本地 Bilibili 设置：

1. 打开 CyberCat，进入 `Settings -> Bilibili`。
2. 填写要使用的 Bilibili URL，例如空间页或收藏夹链接。
3. 使用 `QR Login` 登录，或者直接粘贴已有的 BBDown cookie。
4. 保存设置。CyberCat 会把 cookie 保存在当前本地配置里。

仓库里的辅助脚本和更完整的高级说明见 [apps/service/src/scripts/bilibili/README.md](apps/service/src/scripts/bilibili/README.md)。

## 常用快捷键

- 按住 `z+x` 录音，确认后再发送
- 按住 `x+c` 录音并立即发送
- 按 `Ctrl+Shift+0` 朗读当前选中的文字

## 常见问题

- 如果 Windows 弹出安全提示，在确认你信任该版本后继续安装
- 如果安装后应用没有打开，先重启 Windows 再试一次
- 如果快捷键没有反应，检查是否有其他应用占用了相同按键
